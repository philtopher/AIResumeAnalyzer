Goal:

Develop an AI-powered tool that enables users to convert an uploaded CV from one job role/title to another. The tool should:

Focus on the latest employment entry in the CV, adjusting it to align with a new role specified by the user.

Tailor the CV to address challenges and requirements detailed in a job description supplied via a text box.

Enhance the CV using relevant and recent information about the target organizationâ€™s challenges gathered from Glassdoor reviews, Indeed reviews, articles, and YouTube videos.

Evaluate the revised CV against industry standards, highlighting strengths and weaknesses, and offering actionable advice for improvement.

Provide an option to fully regenerate the CV for the targeted role, ensuring consistency in start and end dates across all employment entries.

Features:

Primary Functionality:

Role Conversion:

Extract the latest employment entry from the uploaded CV.

Rewrite it to align with a new role specified by the user in a text box.

Tailor responsibilities, skills, and achievements to meet the needs of the target job description.

Research & Insights:

Automatically gather and display organizational challenges from:

Glassdoor reviews

Indeed reviews

Articles and blogs

YouTube videos (via scraping or API integration)

CV Evaluation:

Score the generated CV against industry standards.

Highlight areas of strength (e.g., relevant skills, measurable achievements).

Suggest areas for improvement (e.g., insufficient keywords, lack of quantifiable outcomes).

Full CV Generation:

Optionally, regenerate the entire CV for the targeted role, ensuring alignment with the job description provided.

Maintain consistent timelines across all entries.

User Interface:

Input Fields:

Upload CV

Job Description (text box)

New Targeted Role (text box)

Output:

Display the revised CV on the page.

Show research insights from reviews and articles about the organization.

Display CV evaluation results, including:

Strengths

Weaknesses

Recommendations

Options:

Download revised CV.

Download evaluation report.

Security:

User registration with:

Secure password storage (e.g., hashed with bcrypt).

Password reset functionality.

Admin Panel:

Super admin can:

Add/remove sub-admins.

Manually add/remove users.

Sub-admins can:

View user activity logs.

Approve/reject flagged CV edits.
